# Ansible for TB3 setup
## Usage
1. Install [Ansible](https://www.ansible.com/)  
  On ubuntu: `sudo apt install ansible`
1. Clone this repository
1. Edit [hosts](hosts) file and add your hosts
1. Run `ansible-playbook -i hosts site.yml`
